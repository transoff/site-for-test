# Цветочный магазин — одностраничный лендинг

Простой, удобный и полностью статический сайт для цветочного магазина: каталог с изображениями и форма создания заказа. Есть переключение тёмной/светлой темы с сохранением в `localStorage`.

## Что внутри
- Герой-блок с кнопками:
  - «Создать заказ» — плавный переход к форме заказа.
  - «Каталог» — переход к сетке с картинками букетов.
- Каталог: 3 карточки (розы, тюльпаны, ромашки) с изображениями и ценами.
- Форма заказа: имя, телефон, email (необязательно), тип букета, размер (S/M/L), адрес, комментарий, согласие.
- Переключатель темы (луна/солнце) в шапке. Выбор сохраняется между перезагрузками.
- Адаптивная верстка и базовая доступность (aria, правильные подписи полей, фокус-менеджмент).

## Как запустить
1. Откройте файл `site.html` в браузере (двойной клик или через контекстное меню).
2. В шапке нажмите кнопку с иконкой луны/солнца — тема переключится. Обновите страницу — выбранная тема сохранится.
3. На главном экране нажмите «Каталог» — пролистает к каталогу с изображениями.
4. Нажмите «Создать заказ» — пролистает к форме; фокус станет в поле «Имя».
5. Заполните обязательные поля (Имя, Телефон, Тип букета, Согласие) и нажмите «Отправить заказ». Появится зелёное уведомление об успешном создании заказа.

## Примечания по темам
- Тёмная тема: фон `#0f1419`, белый текст, акценты `#1e88e5`.
- Светлая тема: фон `#f5f5f5`, тёмный текст `#333`, акценты `#1e88e5`, границы `#000`.
- Переключение работает без перезагрузки. Выбор сохраняется в `localStorage` с ключом `theme`.
- Для минимизации мерцания неправильной темы используется ранняя инициализация темы в `<head>`.

## Важно
- Сайт статический: отправка формы — демонстрационная, без бэкенда. Данные никуда не отправляются и не сохраняются.
- Изображения каталога загружаются из общедоступного CDN (Unsplash). Для офлайн-режима добавьте локальные изображения и обновите пути.

## Идеи для улучшений
- Реальная отправка заказов на сервер (REST API/почта), уведомление в мессенджер.
- Маска телефона и нормализация номера.
- Динамический каталог (загрузка JSON), фильтры и поиск.
- SEO-улучшения: OpenGraph, фавиконки, микроразметка.
- Покрытие тестами UI-логики (валидация, уведомления) с использованием Jest + jsdom.
