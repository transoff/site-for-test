# site-for-test

## Темная/светлая тема
Добавлен переключатель темы в шапке сайта (луна/солнце) с сохранением в localStorage. По умолчанию загружается тёмная тема.

### Как запустить
1. Откройте файл `site.html` в браузере (двойной клик или через контекстное меню).
2. Кликните по кнопке «Тема» (иконка луна/солнце) в шапке — тема переключится.
3. Обновите страницу — выбранная тема сохранится и применится автоматически.

### Примечания
- Тёмная тема: фон `#0f1419`, белый текст, акценты `#1e88e5`.
- Светлая тема: фон `#f5f5f5`, тёмный текст `#333`, акценты `#1e88e5`, границы чёрные `#000`.
- Переключение работает без перезагрузки. Выбор сохраняется в `localStorage` с ключом `theme`.
- Для минимизации мерцания неправильной темы используется ранняя инициализация темы в `<head>`.
